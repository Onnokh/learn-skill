# learn-skill.sh website

One-page TanStack Start site for `learn-skill.sh`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- Main page route: `src/routes/index.tsx`
- Global styles: `src/styles.css`
- Domain CNAME (for static hosts): `public/CNAME`
